# Smart Hotel Management System

## Description
This project is a DBMS-based hotel management system.

## Database
- schema.sql → table structure
- sample_data.sql → sample data

## Modules
- Customers
- Rooms
- Bookings
- Payments
- Services
- Food Orders
- Feedback
- Housekeeping

## How to use
Import schema.sql and sample_data.sql into MySQL.
